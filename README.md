# Discord Community Bot

サーバーの活気を高めるためのゲーミフィケーション（レベル機能）、匿名チャット、ミーム画像生成、および管理・ログ記録ツールを備えた多機能Discord Botです。

## 機能とコマンド一覧

### 1. レベルシステム (leveling.py)
発言数に応じて自動的にレベルアップし、ユーザーのモチベーションを向上させる機能です。

* **動的ステータス更新**: レベルアップ時に自動でニックネームに [Lv.X] が付与され、特定のレベル（10の倍数など）に到達すると、サーバー内の該当ロール（Lv.10, Lv.20...）が自動的に入れ替わります。

* **!level [@ユーザー]**: 現在のレベルと、次のレベルアップまでに必要なプログレスバーを視覚的に表示します。 

* **!sync_levels**: [管理者用] 過去のチャンネル履歴をすべて探索し、メッセージ数をデータベース（SQLite）に同期・復元します。

* **!sync_show_levels**: [管理者用] サーバーに存在する全メンバーのレベル表示とロールを、現在のデータに基づいて一斉に同期・クレンジング（不要なロールの剥奪）します。

### 2. 匿名チャットシステム (anonymous.py)
DiscordのUI（Modal・Button）を活用した、安全な匿名発言機能です。

* **ハッシュ化ID**: 発言者はユーザーIDに基づく固定の4桁のハッシュID（例：匿名:1234）で識別されるため、完全な匿名性を保ちつつ文脈を追うことができます。

* **!setup_anonymous**: [管理者用] 「匿名会話」という名前のチャンネルに、匿名発言用UIを開くための初期ボタンを設置します。

### 3. ミーム画像生成 (meme.py)
ユーザーの入力テキストを、指定のテンプレート画像内に自動でフィッティング（改行・フォントサイズ調整）して生成・送信します。

* **!dragon <テキスト>**: 「好きな惣菜発表ドラゴン」の吹き出しにテキストを合成します。

* **!robo <テキスト>**: 「マジレスロボ」の画像にテキストを合成します。

### 4. 管理とロギング (admin.py, log.py)

* **!kick <@ユーザー> [理由]**: 対象ユーザーをキックします。

* **!ban <@ユーザー> [理由]**: 対象ユーザーをBANします。

* **VCログ**: メンバーのボイスチャンネルへの入室・退室・移動の履歴を、自動的に「log」チャンネルに記録します（※本番デプロイ用のBot IDでのみ稼働）。

## 動作環境・依存ライブラリ

* **Python 3.x**
* **discord.py (Discord APIラッパー)**
* **aiosqlite (非同期データベース通信)**
* **Pillow (ミーム画像生成用の画像処理)**
